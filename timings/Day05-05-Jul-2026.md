# ⚙️ Linux Process Management

Linux provides commands to monitor and manage running processes.

| Command | Purpose |
|---------|---------|
| `ps` | Show running processes |
| `ps aux` | Display all running processes with details |
| `top` | Real-time system monitoring |
| `pgrep` | Find Process ID (PID) by process name |
| `grep` | Filter command output |
| `egrep` | Search using multiple patterns |
| `|` (Pipe) | Pass output from one command to another |

### Examples

```bash
ps
ps aux
ps aux | grep nginx
ps aux | egrep "nginx|httpd"
pgrep nginx
```

---

# 🛑 Process Termination

Used to stop running processes.

| Command | Purpose |
|---------|---------|
| `kill` | Stop process using PID |
| `kill -9` | Forcefully stop process |
| `kill -15` | Gracefully stop process |
| `pkill` | Stop process using process name |

### Examples

```bash
kill 1234
kill -9 1234
pkill nginx
```

# 📊 Load Average

Load Average shows how busy the system is over time.

| Time | Meaning |
|------|---------|
| 1 Minute | Current system load |
| 5 Minutes | Recent system load |
| 15 Minutes | Overall system load |

```bash
uptime
```

# 💾 Disk & System Monitoring

| Command | Purpose |
|---------|---------|
| `df -h` | Check disk usage |
| `du -sh` | Check directory size |
| `hostname` | Show system hostname |
| `uname -a` | Display system information |
| `whoami` | Show current user |
| `free -h` | Check memory usage |
| `lscpu` | Display CPU information |

### Examples

```bash
df -h
du -sh /home
hostname
uname -a
whoami
free -h
lscpu
```

# ☁️ AWS EC2 EBS Volume Expansion

After increasing an EBS volume in AWS, we also need to extend the partition and filesystem so Linux can use the additional storage.

### Commands

```bash
df -h
lsblk
sudo growpart /dev/xvda 1
sudo resize2fs /dev/xvda1
df -h
```
