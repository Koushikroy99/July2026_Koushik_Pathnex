### Linux Kernel

- Core part of the operating system
- Manages hardware resources: CPU, memory, storage, devices
- Written in C programming language

---

### Shell

- Bridge between user and kernel
- User gives commands to shell → shell passes to kernel

---

### Linux File System

- Everything starts from root directory `/`
- All files and folders are organized under root

---

### Commands Practiced

| Command | Purpose |
|---------|---------|
| `pwd` | Print Working Directory (shows current directory) |
| `ls` | List files and folders |
| `ls -a` | Show hidden files |
| `ls -la` | Show hidden files with details (permissions, owner, size, date) |
| `cd` | Change Directory |
| `cd .` | Current directory |
| `cd ..` | Parent directory |
| `cd ~` | Home directory |
| `cd /` | Root directory |
| `cd -` | Previous directory |
| `mkdir` | Create new directory |
| `touch` | Create new file |
| `cat` | Display file contents |
| `cp` | Copy files |
| `mv` | Move or rename files |
| `find` | Search for files and directories |
| `vim` | Create and edit files |

---

### Hidden Files & Folders

- If a file or folder starts with a dot (`.`), it becomes hidden
- Use `ls -a` to display hidden files
